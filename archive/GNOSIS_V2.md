## The Light Signature

**The core innovation. The thing no one else has. The thing that makes Gnosis worthy of Aether.**

Every human genome produces a unique light signature — a color pattern encoded from their actual SNP data, rendered as a living double helix. No two are alike. Not metaphorically. Mathematically. 3^500 possible patterns from our 500-SNP panel.

### How It Works

**500 SNPs. 500 points of light along the helix.**

Each SNP occupies a fixed position on the helix spiral. The position never changes — MTHFR is always at position 47 (or wherever we assign it). This means two people can look at the same region of their helix and compare.

### Dual Encoding: Shape + Color

The light signature encodes your genome through TWO independent channels — color AND geometry. Either one alone produces recognizable uniqueness. Together they make your helix unmistakable even in silhouette.

#### Channel 1: Color

**Three genotype states → three color channels:**

| Genotype | Encoding | Visual |
|----------|----------|--------|
| Homozygous reference (normal) | Cool channel | Blue-white light, calm |
| Heterozygous (carrier) | Mid channel | Teal-green light, moderate |
| Homozygous variant | Warm channel | Gold-amber-red light, intense |

**Intensity encodes significance:**

| Evidence Grade | Brightness | Behavior |
|----------------|-----------|----------|
| Clinician | Brightest | Pulses slowly, draws the eye |
| Monitor | Medium | Steady glow |
| Lifestyle | Softest | Ambient, background warmth |

**Category determines hue:**

| Category | Hue Family |
|----------|-----------|
| Cardiovascular | Deep red |
| Inflammation | Amber-orange |

| Metabolism | Gold |
| Hormones | Rose |

**SNP positions are scattered** (not clustered by category) — this maximizes visual uniqueness at a glance. Two helixes never look similar from screenshot distance.

**Neither warm nor cool is "better."** A warm helix isn't "bad genetics." It's a complex, interesting genome with more levers to pull. We never pathologize. We illuminate.

#### Channel 2: Geometry

The genome doesn't just color the helix — it SHAPES it. The helix structure itself is driven by your data.

| Parameter | Driven By | Effect |
|-----------|-----------|--------|
| **Twist rate** | Overall variant density | More variants = tighter, more complex coils. Fewer = open, flowing spirals. |
| **Amplitude** | Cognitive/neuro profile (COMT, BDNF, DRD2, SLC6A4) | High activity = wider, more expressive oscillation. Low = narrow, contained. |
| **Strand thickness** | Evidence grade along the helix | Clinician-grade regions = thick, glowing strands. Clean stretches = thin, ethereal. |
| **Asymmetry** | Methylation profile (MTHFR, MTR, MTRR) | Balanced methylation = symmetrical strands. Impaired = one strand dominant over the other. |
| **Rungs/bridges** | Gene-gene interaction density | Many interactions = dense, interconnected bridges. Independent findings = open space between strands. |

**The result:** Every helix is unique in shape, proportion, AND color simultaneously. You'd never mistake yours for someone else's even in silhouette. Some helixes are tight, dense, and warm. Others are open, flowing, and cool. Some are asymmetric with thick clusters. Each one is a portrait.

**During the Genesis transformation (Scene 1),** it's not just colors flooding in — the helix itself RESHAPES. The generic landing page helix morphs as strands tighten or loosen, bridges form, thickness shifts, amplitude changes. The structure becomes yours.

## Technical Architecture

### SNP → Light Encoding Pipeline
```
User uploads .txt/.csv
  → Parse (existing report-engine.ts)
  → 500 SNPs resolved with genotype + evidence grade + category
  → SNP Position Map (fixed, versioned)
  → Color encoding: genotype → channel, category → hue, evidence → intensity
  → Light Signature JSON: Array<{ position: number, color: [r,g,b], intensity: number, pulse: boolean }>
  → Stored in Supabase (user_id → light_signature)
  → Three.js helix renderer receives signature and transforms
```

### Helix Renderer (Three.js)
- Existing HelixCanvas.tsx as base
- New: accepts `lightSignature` prop
- Each particle/point on helix mapped to SNP position
- Color/intensity applied per-point
- Transformation animation: generic → personalized (3-4s bloom)
- Post-processing: existing bloom + edge glow pipeline
- Export: canvas → PNG for sharing (1200x630 social card)

### Archetype Derivation
```
Light signature → frequency analysis
  → Dominant category clusters
  → Variant density profile
  → Cognitive/behavioral gene weighting
  → Archetype matrix lookup
  → Name + description + tagline
```

### Persistence
- Light signature stored with user record
- Versioned: if panel expands (501+), signature grows but never changes existing positions
- Exportable: user can download their signature as JSON or image
- API: `GET /api/signature/:userId` returns signature for other Hypostas products
